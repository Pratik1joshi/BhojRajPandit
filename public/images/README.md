# Image Setup Instructions

## Required Image

Please save the Pandit's photo as:
`pandit-photo.jpg`

**Location:** Place it in this folder (`public/images/pandit-photo.jpg`)

**Image Details:**
- The image should be the kayaking photo (man in orange life jacket)
- Recommended dimensions: 800x1000 pixels or similar portrait orientation
- Format: JPG or JPEG
- File size: Less than 500KB for optimal loading

## How to Add the Image

1. Save the attached image from the conversation
2. Rename it to `pandit-photo.jpg`
3. Place it in `public/images/` folder
4. The website will automatically display it on:
   - Home page (About section)
   - About page

The image is already configured in the code with proper responsive sizing and Next.js Image optimization.
